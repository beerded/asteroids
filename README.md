# Asteroids

This is a classic arcade game written in python.

To run, type `uv run main.py`

The following are the control keys:
- **A**: rotate craft left
- **D**: rotate craft right
- **W**: move craft forwards
- **S**: move craft backwards
- **\<Spacebar\>**: shoot

Goal is to shoot as many asteroids as possible while avoiding getting hit. Game ends when your craft collides with an asteroid
